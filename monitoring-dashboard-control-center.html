🔍 SENTRY ERROR TRACKING SETUP

Step 1: Create Sentry Project

  1. Go to sentry.io/signup
  2. Create new Node.js project
  3. Copy your DSN (format: https://XXXXXX@XXXXX.ingest.sentry.io/XXXXX)

Step 2: Add to Render Environment

SENTRY_DSN= [Your DSN from Step 1]

Step 3: Verify

Errors will appear in your Sentry Dashboard automatically

📊 GOOGLE ANALYTICS 4 SETUP

Step 1: Create GA4 Property

  1. Go to analytics.google.com
  2. Create new property: "Aetra-Production"
  3. Add web stream: https://freight-flow.netlify.app
  4. Copy Measurement ID (format: G-XXXXXXXXXX)

Step 2: Add to Netlify Environment

REACT_APP_GA4_ID= [Your Measurement ID from Step 1]

Step 3: Verify

Check GA4 Real-time Report - you should see your session within 30 seconds
